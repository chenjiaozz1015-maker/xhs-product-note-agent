# 种草机

## 1. 项目简介
种草机是一个面向小红书内容发布者、普通好物分享用户、小商家、宝妈以及初级博主的轻量 MVP 网页工具。用户上传商品照片和一句描述后，系统会自动生成小红书风格封面图、配图和发布文案。

## 2. 目标用户
- 小红书内容发布者
- 普通好物分享用户
- 小商家
- 宝妈
- 探店 / 测评 / 初级博主

## 3. MVP 功能范围
- 上传 1 张或少量商品图片
- 输入一句商品描述
- 选择内容类型和风格
- 生成 3 张模板化图片
- 生成 5 个标题、1 篇正文、10 个标签、3 条评论引导
- 支持下载图片和复制文案

## 3.1 v0.1-2 当前已实现
- 上传商品图
- 生成 3 张小红书风格图片
- 生成标题、正文、标签、评论引导
- 上传图片预览
- 点击大图预览
- 复制成功提示
- 下载单张图片
- 下载全部图片，前端触发多文件下载
- 移动端适配

## 4. 技术栈
- Python 3
- FastAPI
- Jinja2
- 原生 HTML / CSS / JavaScript
- Pillow
- 本地目录保存上传图片和生成图片

## 5. 安装依赖方式
```bash
pip install -r requirements.txt
```

## 6. 启动方式
```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 7. 访问地址
访问 http://127.0.0.1:8000/
