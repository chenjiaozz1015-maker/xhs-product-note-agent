# 种草机（xhs-product-note-agent）

种草机是一个本地 MVP 网页工具：上传商品照片，填写商品名称、商品类目和一句描述后，生成小红书风格图片素材和发布文案。当前不接大模型、不做图片识别，主要通过类目化模板提升内容相关性。

## 当前已实现
- 上传商品图片
- 商品图片预览
- 填写商品名称
- 选择商品类目
- 填写一句商品描述
- 选择内容类型和风格
- 根据商品类目生成更相关的标题、正文、标签、评论引导
- 生成 3 张小红书风格图片
- 点击大图预览
- 下载单张图片
- 下载全部图片，前端触发多文件下载
- 复制标题 / 正文 / 标签 / 评论引导 / 全部发布文案
- 中文字体渲染修复
- 基础错误提示
- 移动端适配

## 技术栈
- Python 3
- FastAPI
- Jinja2
- 原生 HTML / CSS / JavaScript
- Pillow

## 本地启动
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
py -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

访问：

```text
http://127.0.0.1:8000/
```

## 测试
```bash
py -m pytest -q
```

## 运行文件说明
运行时上传图片和生成图片会写入：

- `app/static/uploads/`
- `app/static/generated/`

这些运行文件已通过 `.gitignore` 排除，目录中的 `.gitkeep` 可保留进入 Git。
